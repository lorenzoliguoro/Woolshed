# Woolshed Design Marathon Idea Capture Form

This repository hosts a standalone HTML form developed for the Woolshed project, co-funded by the European Union through the Interreg Alpine Space Programme 2021–2027.

The tool supports the documentation of ideas generated within the Design Marathons carried out under Work Package 2, Action 2.2. Its purpose is to collect, organise and export information on concepts, material experiments, samples and prototypes emerging from the participatory design activities.

## Project context

Woolshed: A New Era for Alpine Wool addresses the underuse of coarse and local wool in the Alpine area by exploring new applications, materials, products and value-chain opportunities.

Within WP2, the Design Marathons are intended to involve designers, makers, artists, researchers, wool producers, processors and other territorial actors in the generation of new wool-based ideas. This form has been designed as a practical documentation tool for these activities.

The form focuses only on what directly emerges from Action 2.2. It does not replace the following project activities dedicated to technical development, market analysis, business model validation or uptake support.

## What the form does

The HTML form allows users to collect structured information about each idea, including:

- idea title, authors and contact person;
- countries where the idea could initially be developed;
- type of contributors involved;
- idea category and possible field of use;
- current form of the idea, from initial intuition to prototype;
- wool material used or imagined;
- making techniques and process notes;
- intended users and use contexts;
- immediate continuation needs and facilitator notes.

The form combines dropdown menus, multi-select fields and short open-text fields. This makes the data easier to compare across different Design Marathons while still allowing participants and facilitators to describe each idea in their own words.

## Main features

The tool works directly in the browser and does not require a login, database or server-side application.

Users can:

- fill in one idea at a time;
- add multiple ideas to a local collection;
- review the collected records in a table;
- edit or delete records before export;
- download the collected data as a CSV file;
- generate a print-ready version that can be saved as PDF;
- use the form from desktop, tablet or mobile browsers.

Data are stored locally in the user’s browser. They are not automatically transmitted to GitHub, to the project partners, or to any central database.

## Recommended workflow

The recommended workflow is simple:

1. Open the online form in a browser.
2. Fill in the form for one idea.
3. Click “Add idea to collection”.
4. Repeat the process for all ideas generated during the session.
5. Review the table of collected ideas.
6. Export the data as CSV and/or PDF.
7. Upload the exported files to the shared Woolshed project folder or send them to the responsible WP2 coordination team.

## Data management note

This tool is intended for documentation and internal project follow-up. Since data are stored locally in the browser, each partner or facilitator is responsible for exporting and sharing the completed records.

If the browser cache is cleared, locally stored records may be lost. For this reason, users should export the CSV file immediately after completing a session.
